### index.js Random출력 설명
```
자식클래스에서 발표자 2명 이상을 입력받는다. (process.argv.slice(2))
부모 클래스에서 print함수를 이용해서 출력하도록 한다. (print())
Math.random함수를 이용해 랜덤으로 발표자를 출력하도록 한다.
** Math.random함수는 0.0~1.0부터 실수값을 출력하기에 소수값을 버려주는 Math.floor함수도 이용**
```
